# Customer-Review-Analysis
Customer review analysis using TF-IDF and MTNN for rating prediction and recommendation classification.

Project Description : This project focuses on analyzing customer reviews using a multi-task learning approach. The system processes unstructured textual data through preprocessing techniques and TF-IDF feature extraction. A Multi-Task Neural Network (MTNN) is used to perform both rating prediction (regression) and recommendation classification simultaneously. The model is further enhanced using Extra Trees CART to improve prediction performance and stability.
#Features: Text preprocessing and data cleaning, TF-IDF based feature extraction, Multi-task learning (MTNN), Rating prediction (Regression), Recommendation classification (Yes/No), Model comparison and evaluation, High accuracy and reliable predictions.
#Technologies Used : Python, Machine Learning, TF-IDF, Multi-Task Neural Network (MTNN), Extra Trees CART, Flask (for deployment).
#Dataset : The dataset contains customer reviews with features like; Review Text, Rating, Recommended Indicator, Product-related attributes.
#Results: The proposed model (MTNN + Extra Trees CART) achieved high accuracy and balanced performance across precision, recall, and F1-score, demonstrating reliable predictions.
